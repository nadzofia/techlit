COMPUTER PROGRAMMING

THE C LANGUAGE

Lesson, February 11, 2023

C-language is a general-purpose and procedural-oriented programming language.
It is a structured and machine-independent programming language. It was developed by Dennis Ritchie in 1972 at the AT&T Bell Laboratories. It was developed along with the UNIX operating system, and is strongly linked with the UNIX operating system. History of C language revolves around development as a system implementation language to write an operating system.
The basic syntax of the C program consists of the header, main() function, variable declaration, body, and return type of the program.

- HEADER "#include " - the header is the first line in the C program with extension .h which contains macro definitions and C functions.
- MAIN int main() - programs must contain main() function, because execution in C programming starts from main().
- Variable declaration in C is done inside the main function and can be used in the body anywhere but before the main function, we can also declare variables which are known as Global variables.
- In the body of the function, we perform operations required inside the function like printing, sum, average, sorting etc.
- The last part of the C program is the return statement which refers to returning values of the program. If the return type is void then there will be no return statement.
- Semicolon is used to show the termination of an instruction.

SAMPLE CODE:
#include < stdio.h >
int main() {
printf("Hello World!");
return 0;
}
